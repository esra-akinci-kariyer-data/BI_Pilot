# BI_Pilot

A Streamlit-based Proof of Concept (PoC) for report metadata exploration and business intelligence support.

## Live Demo
[Open the app](https://esraakinci.streamlit.app/)

## Overview
This project was developed as a PoC to explore how report metadata can be organized, searched, and presented through a simple interactive interface.

## Features
- Interactive Streamlit interface
- Report metadata exploration
- CSV-based data handling
- Lightweight BI-oriented PoC structure

## Tech Stack
- Python
- Streamlit
- Pandas
- CSV

## Project Structure
- `streamlit_app.py` – main Streamlit application
- `app.py` – application logic
- `ui_app.py` – UI-related functions
- `requirements.txt` – dependencies
- `.github/workflows` – deployment workflow

## Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
